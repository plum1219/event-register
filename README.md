# イベントレジ 試作版 v2

ブラウザだけで動く、同人イベント向けレジ・在庫管理ツールのたたき台です。

## v2で追加・修正した機能

- 価格は100円刻み以外も入力可能
  - 入力欄の上下ボタンは100刻み
- アイコン色の変更
- 自分用確保分の入力
  - デフォルトは1
- 在庫総数の注意文
- 商品の並び替え
- 商品修正ボタン
- 通販用・自分用の後から変更
- 通販用・自分用・取り置き数が残り在庫に反映
- 残り部数を大きく表示
- 販売済み・取り置き・通販用・自分用を別表示
- 計算用レジ
- 受け取り額とおつり計算
- 取り置きリスト内の合計金額表示
- 取り置き内容を会計に読み込み
- 取り置き点数の後から編集
- 取り置き分を在庫から差し引き

## 在庫計算

通常頒布の残り部数 = 在庫総数 − 販売済み − 取り置き − 通販用 − 自分用

## 使い方

ZIPを解凍して `index.html` をブラウザで開いてください。
データは同じブラウザの localStorage に保存されます。
バックアップは「データを書き出し」からJSONで保存できます。

## 注意

これは初期試作です。実イベントで使う前に、スマホ・タブレットでの操作性、誤操作防止、バックアップ方法を確認してください。
