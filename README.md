# イベントレジ 試作版 v4

ブラウザだけで動く、同人イベント向けレジ・在庫管理ツールのたたき台です。

## v4で追加・修正した機能

- 「販売」表記を「頒布」表記に変更
- 「イベント中」と「事前準備」のタブに分割
- 事前準備側に、サークル設定・商品登録・取り置き登録・データ管理を配置
- レジに「在庫に反映して計算をクリア」ボタンを追加
- レジに入力した頒布数を在庫管理の頒布済みに反映可能
- 完売告知ボタンは、残り部数が0になるまでタップ不可
- スマホではレジの「頒布価格の合計」「受け取った額の合計」を横並び表示
- レジの「おつり」を目立つ表示に変更

## 在庫計算

通常頒布の残り部数 = 在庫総数 − 頒布済み − 取り置き − 通販用 − 自分用

## 使い方

`index.html` をブラウザで開いてください。データは同じブラウザの localStorage に保存されます。
