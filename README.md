# イベントレジ 試作版

ブラウザだけで動く、同人イベント向けレジ・在庫管理ツールのたたき台です。

## 使い方

1. `index.html` をブラウザで開きます。
2. 商品を登録します。
3. 「+1 販売」で販売数と売上が更新されます。
4. 在庫が0になると、完売告知ポップアップが出ます。
5. データは同じブラウザの localStorage に保存されます。
6. 「データを書き出し」でJSONバックアップできます。

## 入っている機能

- 商品登録
- アイコン選択
- 価格・在庫・通販用取り分の管理
- 販売カウント
- 誤タップ取り消し
- 売上合計
- 取り置きリスト
- 受け渡し済みチェック
- 完売時ポップアップ
- X投稿画面への遷移
- 告知待ちアラート
- JSON書き出し / 読み込み

## 注意

これは初期試作です。
実イベントで使う前に、スマホ・タブレットでの操作性、誤操作防止、バックアップ方法を確認してください。
